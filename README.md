WFAT README
===

WFAT is the Web Forrest Analysis Toolkit designed to help analyse a
large number of web sites for redirection structure, configured
headers etc.

Installation
---

Type "ant".


Normal usage
---

For options run

	java -jar wfat-0.6.jar -?
